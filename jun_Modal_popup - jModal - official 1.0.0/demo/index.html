jun_modal_popup demo

class="modal"로 한 경우

직계 a태그의 img태그 → modal popup (O)
a태그 없는 직계 img태그 → modal popup (X)
직계 a태그의 img태그 → modal popup (O)
후손에 속한 a태그 → 모달 팝업이 아닌 새창(new window)으로 열림
a태그 없는 후손 img태그

후손 a태그의 img태그 → modal popup (O)
a태그 없는 후손 img태그
